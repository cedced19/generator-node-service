## Required

Of course Node.js, NPM and Git.

Yeoman:

```bash
$ npm install -g yo
```

Bower:

```bash
$ npm install -g bower
```

Generator-Node-Service:

```bash
$ npm install -g generator-node-service
```


## Getting Started

Initiate the generator:

```bash
$ yo node-service
```

To launch in developpement:

```bash
$ npm start
```

To launch in release:

```bash
$ grunt
$ cd dist/
$ npm start
```
